Setup Guide

Connect to Shortcuts
Each category needs one iOS Shortcut to receive the photo and save it to your vault. Set this up once per category — after that, logging is just a few taps.
1
Create a new Shortcut
Open the Shortcuts app on your iPhone. Tap + in the top-right corner to create a new shortcut.
2
Add "Get Latest Photos"
Tap Add Action, search for Get Latest Photos, and add it. Tap the action to expand it and set the count to 1. This grabs the photo you just took.
3
Add "Set Name"
Tap +, search for Set Name, and add it. Tap the Name field, then tap Insert Variable and choose Shortcut Input. This applies the filename sent by this app to the photo.
4
Add "Save File"
Tap +, search for Save File, and add it. Tap the destination field to open the folder browser — navigate to your vault folder for this category:
On My iPhone → Obsidian → [Your Vault] → [destination folder]
Turn off Ask Where to Save so it saves silently every time.
Your destination folders are listed in the shortcut table above.
5
Name your Shortcut exactly
Tap the shortcut name at the top to rename it. The name must match exactly — including capitalisation. One shortcut per category:
6
Test it!
Tap a category in this app, take a photo, confirm the preview, and tap Log It. The shortcut will run and save the photo to your vault folder.
The first time it runs, iOS may ask for Photos access — tap Allow.
Note: iOS does not allow apps to create Shortcuts automatically. You only need to do this setup once per category — after that, logging is just a few taps.
